GATE Online Examination Instructions

General Instructions

  • The clock will be set at the server. The countdown timer at the top right of the screen will display the remaining time available for you to complete the examination.
  • When the timer reaches zero, the examination will end by itself. You will not be required to end or submit your examination.
  • The Question Palette displayed on the right side of the screen will show the status of each question using one of the following symbols:
    1 Not Visited
    2 Not Answered
    3 Answered
    4 Marked for Review
    5 Answered & Marked for Review (will be considered for evaluation)

Question Types & Marking Scheme

Question Type Description Correct Response Incorrect Response
MCQ (Multiple Choice) Only one option is correct out of four. +1 or +2 Marks -1/3 or -2/3 Marks (Negative)
MSQ (Multiple Select) One or more options can be correct. Must select all correct and no incorrect options. No partial marking. +1 or +2 Marks 0 Marks (No negative marking)
NAT (Numerical Answer) Enter a numerical value using the virtual keypad. +1 or +2 Marks 0 Marks (No negative marking)
